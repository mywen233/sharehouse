ShareHouse, is a community or campus-oriented web application designed for sharing idle items, managing second-hand trading, and facilitating community activities like Lost & Found and Book Sales. It is built using a classic Java stack: Struts2 + Spring + Hibernate (SSH).

project_summary.md

📋 Key Functionality Overview
The platform is divided into several modules:

👤 User Management: Registration, Login (with credentials and auto-login cookies), Email Activation, Password Reset, and Social Binding (RenRen integration).
📦 Idle Goods Sharing: Publishing items, tracking images, listing with pagination, and managing transactions/orders.
🏘️ Activities & Campus-Specific Features:
Lost and Found: Tracking posting of lost and found objects.
Book Exchanging: A marketplace dedicated to buying or selling textbooks.
💬 Messaging & Social Interactivity: Peer-to-peer private messages, comments on shares/goods with threading algorithms, notifications, and @ mentions.
🛠️ Utilities Layer Content Filter: Anti-spam sensitive words filtration using WordsFilter, standard Pinyin conversions, dynamic image rescaling engines.

(From the author: This is my university project. It’s a campus social platform where students can share, buy, and sell second-hand items. It features dedicated communities (or circles) for different universities, faculties, and grade levels. Users can post item updates, comment, repost, like, add friends, and send direct messages.)
